GCP Certifications

Here follows advice for anyone taking a Google Cloud certification for the first time. I recently took (and passed): Google Cloud Certified Professional Data Engineer Google Cloud Certified Professional Cloud Architect Both exams are two hours long, posing 50 questions in a multiple choice/select format with four possible answers. At the end you’re given a provisional pass or fail. You’re not told what the pass threshold is or how many questions you answered correctly.
